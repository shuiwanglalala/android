# SeekBar

```xml
<SeekBar
        android:id="@+id/local_seekbar"
        android:layout_width="1692dp"
        android:layout_height="wrap_content"
        android:paddingEnd="36dp"
        android:paddingStart="36dp"
        android:progressDrawable="@color/transparent"
        android:thumb="@drawable/local_seekbar_thum_bg"
        android:thumbOffset="36dp" />
```

SeekBar有默认padding和thumboffset，注意！
