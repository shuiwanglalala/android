# 应用层

+ 应用层概述
  + C/S
    + 服务器
      + 永久提供服务
      + 永久性访问地址/域名
    + 客户端
      + 与服务器通信，使用服务器的服务
      + 间隙性接入网络
      + 可能使用动态IP
      + 不与其他客户机自己通信
  + P2P
    + 不存在永远在线的服务器
    + 每个主机既可以提供服务，也可以请求服务
    + 任意节点之间可以通信
    + 节点间隙性接入网络
    + 节点可能改变IP地址
    + 可扩展性好
    + 网络健壮性好
+ DNS
  + 域名
    + 顶级域名
      + 国家顶级域名 cn us uk
      + 通用顶级域名 com net org gov int aero meseum travel
      + 反向域名 arpa
    + 二级域名
      + 类别域名 ac com edu gov mil net org
      + 行政区域名
  + 域名服务器
    + 根域名服务器
    + 顶级域名服务器
    + 权限域名服务器
    + 本地域名服务器 高速缓存
  + 持续发送
    + 递归查询 
    + 迭代查询

+ FTP 提供不同种类主机系统（软/硬件系统都可以不同）之间的文件传输能力
