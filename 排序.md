直接插入排序

```java
void InsertSort(int[] A) {
    int i,j;
    for(i=2;i<=A.length;i++) {
        int temp=A[i];
        for(j=i-1;tmp<A[j];j--) {
            A[j+1]=A[j];
        }
        A[j+1]=temp;
    }
}
```
