1.2

冯.诺依曼计算机的特点

+ 计算机由五大部件组成
+ 指令和数据以同等地位存于存储器，可按地址寻码
+ 指令和数据用二进制表示
+ 指令由操作码和地址码组成
+ 存储程序（最重要的特点）
+ 以运算器为核心

存储器的基本组成

+ 存储体

+ MAR 存储器地址寄存器 反应存储单元的个数
+ MDR 存储器数据寄存器 反应存储字长 

1.3

计算机硬件的主要技术指标

+ 机器字长 CPU一次能处理数据的位数，与CPU中的寄存器位数有光
+ 运算速度
  + 主频
  + 核数，每个核支持的线程数
  + CPI 执行一条指令所需时钟周期数
  + MIPS 每秒执行百万条指令
  + FLOPS 每秒浮点运算次数
+ 存储容量
